# Java Coding Practice

This repository contains a curated collection of Java coding problems focused on logic building, problem solving, and core Java concepts. The problems are organized by difficulty to show a clear progression from fundamentals to advanced logic.

Each program is self-contained, cleanly written, and runnable, making this repository suitable for practice, revision, and interview preparation.

## Repository Focus

- Core Java syntax and control flow
- Logical reasoning and edge case handling
- Pattern printing and matrix-based problems
- String manipulation and number-based logic
- Object-Oriented Programming concepts
- Real-world inspired problem statements

The emphasis is on writing readable, structured, and maintainable Java code.

## Folder Structure

Basic-Java-Logic-Programs/  
Intermediate-Java-Logic-Programs/  
Advanced-Java-Logic-Programs/  

### Basic Java Logic Programs
Covers beginner-friendly problems involving conditions, loops, strings, numbers, and basic OOP concepts. Ideal for building strong fundamentals.

### Intermediate Java Logic Programs
Includes problems that require deeper reasoning, pattern generation, matrix operations, and better code structuring.

### Advanced Java Logic Programs
Focuses on advanced logic, interfaces, complex string and array handling, and real-world style scenarios.

## How to Run

Compile the program:
```
javac FileName.java
```

Run the program:
```
java FileName
```

Each file is independent and does not rely on external libraries.
